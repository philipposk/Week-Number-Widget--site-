Technical Details

For developers and curious minds

üì± iOS Implementation

  • Built with SwiftUI and WidgetKit
  • Requires iOS 17+ and Xcode 15+
  • ISO 8601 week calculation with optional Sunday-start mode
  • Customizable colors, label visibility, and week start day
  • Home screen widget with automatic updates

ü§ñ Android Implementation

  • Built with Kotlin in Android Studio
  • Min SDK 26 (Android 8.0), Target SDK 34
  • Home screen and lock screen widget support
  • Automatic dark/light theme based on system settings
  • ISO 8601 standard week calculation
